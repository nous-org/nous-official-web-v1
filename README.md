# NOUS Website

Official website for NOUS, an AI transformation and technology company focused on helping organizations build, deploy, and adopt intelligence in real work.

## Stack

- Astro
- React
- Tailwind CSS
- Cloudflare Workers adapter
- Clerk for admin authentication
- Turso/libSQL for content and newsletter storage
- Resend for contact e-mail delivery

## Local Development

```bash
pnpm install
pnpm dev
```

The default local URL is `http://localhost:4321`.

## Validation

```bash
pnpm typecheck
pnpm lint
pnpm test
pnpm build
pnpm audit:prod
```

`pnpm lint` currently runs `astro check`, so it validates Astro and TypeScript diagnostics.

## Environment

Copy `.env.example` for local placeholders. Production secrets must be configured in the deployment environment, not committed to the repository.

Required runtime integrations include:

- `CLERK_PUBLISHABLE_KEY`
- `CLERK_SECRET_KEY`
- `CLERK_ADMIN_USER_IDS` or an equivalent admin authorization policy
- `TURSO_DATABASE_URL`
- `TURSO_AUTH_TOKEN`
- `TURSO_NEWSLETTER_URL`
- `TURSO_NEWSLETTER_TOKEN`
- `RESEND_API_KEY`

## Notes

- `/admin` is noindexed and requires explicit Clerk admin authorization.
- `/api/contact` and `/api/subscribe` are the active public form endpoints.
- The sitemap is generated by Astro at build time.
